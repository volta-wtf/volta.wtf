# About This Template

This is a documentation template built with modern web technologies.

## Technology Stack

- **Next.js** - React framework for production
- **Fumadocs** - Documentation framework
- **MDX** - Markdown with JSX components
- **Tailwind CSS** - Utility-first CSS framework

## Purpose

This template is designed to help you quickly set up a professional documentation site for your projects. It includes all the essential features you need:

- Clean and modern design
- Easy content management with MDX
- Built-in components for documentation
- SEO optimized
- Performance focused

## Customization

You can customize this template by:

1. Modifying the content in the `content/` directory
2. Updating styles in the `styles/` directory
3. Customizing components in the `components/` directory
4. Configuring settings in `source.config.ts`
