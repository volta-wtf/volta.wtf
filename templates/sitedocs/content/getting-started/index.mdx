# Getting Started

Welcome! This guide will help you get up and running with your documentation site.

## Overview

This documentation template uses:

- **Fumadocs** for the documentation framework
- **MDX** for content authoring
- **Next.js** for the application framework

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or later
- pnpm, npm, or yarn package manager
- A code editor (VS Code recommended)

## Quick Start

Follow these steps to get started:

1. Install dependencies
2. Configure your site
3. Add your content
4. Deploy

Continue to the [Installation](/docs/getting-started/installation) guide for detailed instructions.
