**ToggleGroup** — segmented set of toggles. `type="single"` behaves like a radio group; `type="multiple"` allows several.

```jsx
<ToggleGroup type="single" variant="outline" defaultValue="center">
  <ToggleGroupItem value="left" aria-label="Left"><Icon name="back" /></ToggleGroupItem>
  <ToggleGroupItem value="center" aria-label="Center"><Icon name="menu" /></ToggleGroupItem>
  <ToggleGroupItem value="right" aria-label="Right"><Icon name="forward" /></ToggleGroupItem>
</ToggleGroup>
```
