Standard action button; use `primary` once per view, `secondary`/`ghost` for the rest.

```jsx
<Button>Publish</Button>
<Button variant="outline" size="sm">Export as PNG</Button>
<Button variant="ghost"><Icon name="add" size={14} /> Add layer</Button>
```

Variants: primary · secondary · outline · ghost · destructive · link. Sizes: sm 24px · md 28px · lg 32px. Copy: sentence case, short imperative ("Add layer", never "Click here!").
